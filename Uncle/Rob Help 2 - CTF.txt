Things that you should be able to do:
* exploit the Damn Vulnerable Web Application http://www.dvwa.co.uk/
* create write-ups for everything that you exploit
* build your own tools as required (shell scripting, python, C)
* explain a vulnerability to people that have a computer background, but are not in the security field
* create an exploit using information from a CVE
* create a VM environment from scratch (KVM / virtual box / vmware, linux machines, windows machines, networking)
